Listas Ordenadas

Por padrão a lista ordenada vai iniciar com a numeração 1

  1. Acordar
  2. Escovar os dentes
  3. Ligar para o João
  4. Tomar café

Mas com o uso do TYPE na tag ol podemos alterar para A, a, I, i

  1. Acordar
  2. Escovar os dentes
  3. Ligar para o João
  4. Tomar café

  2. Escovar os dentes
  3. Ligar para o João
  4. Tomar café

  4. Tomar café

  2. Escovar os dentes
  3. Ligar para o João
  4. Tomar café

Também podemos iniciar a partir de um determinado número ou letra, para isso adicionamos o START na tag ol

  1. Acordar
  2. Escovar os dentes
  3. Ligar para o João
  4. Tomar café

OU

  1. Acordar
  2. Escovar os dentes
  3. Ligar para o João
  4. Tomar café

Listas Não Ordenadas

Por padrão a lista não ordenada vai iniciar com um dot no início de cada ítem

  • Arroz
  • Feijão
  • Macarrão

Podemos alterar ess dot por um circulo com o Type dentro da tag ul

  • Arroz
  • Feijão
  • Macarrão

Podemos alterar ess dot por um quadrado com o Type dentro da tag ul

  • Arroz
  • Feijão
  • Macarrão
